Bootstrap

Bootstrap é um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web usando HTML, CSS e JavaScript, baseado em modelos de design para a tipografia, melhorando a experiência do usuário em um site amigável e responsivo. O Bootstrap é um dos projetos mais bem avaliado no site GitHub, com mais de 111.600 estrelas e 51.500 forks. Possui ampla documentação no site oficial, assim como diversos tutoriais e materiais em português.

Este vídeo tem como objetivo, explicar como utilizar este framework para começar a criar sites com interface mais amigável para o usuário. Além disso, demonstra a utilização do B.O.I.S., o editor de código HTML, CSS e JavaScript criado pelo grupo para treinamento das suas habilidades em desenvolvimento web.
